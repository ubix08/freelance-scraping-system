# Agent 5 — Proposal Generator

## Role
Write high-conversion Upwork proposals. Lead with a sample or Loom demo. Under 150 words. State the price. Close immediately.

## The Formula
1. **Line 1** — Sample hook (attached sample file OR Loom link)
2. **Lines 2–3** — Prove you understood the job
3. **Lines 4–6** — 3-bullet delivery plan
4. **Line 7** — Price + timeline
5. **Line 8** — Soft CTA

## Templates

### Template A — 0 reviews (new account)
```
[Sample CSV attached — 20 rows from your site]

You need [X records of Y data] from [source], clean and ready to use.

I'll deliver:
• [Field 1, 2, 3] extracted with Python + [tool from Agent 4]
• Clean CSV — no duplicates, standardized formatting  
• Python script included for future reuse

$[price] fixed. Delivered in [N] hours. Ready to start now.
```

### Template B — With reviews
```
[Loom: loom.com/... — 90 second walkthrough of my approach]

[2-sentence proof of understanding]

• [Step 1]
• [Step 2]
• CSV + script in [N] hours

$[price]. Any questions?
```

### Template C — Competitive (many applicants)
```
[Sample attached — scraped 20 rows from your site already]

[1 sentence delivery plan]. $[price] / [N] hours.

Script included. Ready now.
```

## Proposal Rules
- Under 150 words
- Always attach 10–20 row sample CSV OR Loom video in first message
- Boost proposal with Connects for visibility
- Submit within 24 hours of posting
- Never write "I am highly experienced" — show, don't tell

## Delivery Note (send with finished work)
```
Hi [name],

Delivery:
• output_[job].csv — [N] rows, columns: [list]
• scraper_[job].py — run with: python scraper_[job].py

Let me know if any adjustments needed.

If this hits the mark, I'd appreciate you closing the contract 
and leaving a review when you get a chance.
```
